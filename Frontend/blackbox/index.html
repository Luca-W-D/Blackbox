About The Bot

BlackBox is the ultimate tool of any Starscape trader. 3 in-game bots work around the clock to provide up-to-date market information for Citadel and Harbor. Users can query the most recent data, or request notifications about their market orders.

Data

BlackBox provides data at an unprecedented rate. Resources are refreshed in less than 5 minutes, while all other items for sale at Citadel are refreshed nearly every 90 minutes. Over 150 unique items are tracked every single hour, and 2,000 users total have been indexed by the bot.

Notifications

BlackBox's Discord direct messages keep traders engaged with the game regardless of their availability. Find out when you're top of the list, when an item sells, or when you're order is completed -- all while touching grass.

Morality

BlackBox's in-game bots use Python scripts to interact with the game, manipulating the GUI to take screenshots and run optical character recognition. There is no part of the system that uses any form of code-injection, meaning the only advantage BlackBox has over other traders is time. No other system of its kind exists; hours have been spent to create a system that relies on vanilla game interactions instead of synapsing endpoints.
